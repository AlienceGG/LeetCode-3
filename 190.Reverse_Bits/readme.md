## 190. Reverse Bits (Easy)

### **链接**：
题目：https://leetcode.com/problems/reverse-bits/  
代码(github)：https://github.com/illuz/leetcode

### **题意**：
将一个 32 位无符号的数的二进制反转。

### **分析**：

1. 正常都是想先转 string 再转回去
2. 或者用位运算来反转
3. C++ Trick: 用 `bitset`
4. Python Trick: 用 `bin()` 和 `zfill()`
5. Java Trick: 用 `Integer.reverse()`
