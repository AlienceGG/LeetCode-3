## 150. Evaluate Reverse Polish Notation (Medium)

### **链接**：
题目：https://leetcode.com/problems/evaluate-reverse-polish-notation/  
代码(github)：https://github.com/illuz/leetcode

### **题意**：
求一个逆波兰式的值。

### **分析**：

很经典的栈应用。  
对字符串的处理，Python: `int()`，C++/C: `atoi()`，Java: `Integer.parseInt()`。  
需要注意的是 Python 的 `/` 操作。
